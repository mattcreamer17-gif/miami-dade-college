# Miami Dade College: Landing Page

A single-page working note prepared for Miami Dade College leadership, framed as a read on where AI actually fits in MDC's stack and where it does not. The page mirrors MDC's institutional register (Open Sans, #005dab blue, white background, modest radii) and carries the Section 7 pitch from the fit analysis end-to-end: the 60/30/10 frame, the Edinburgh/UKICER case study, the ICM paper as the underlying framework, and a 30-minute call with Matt Creamer as the next step.

Intended URL path: `/for/miami-dade-college`

## Files

- `index.html` — the landing page
- `styles.css` — brand-native styling (no external CSS framework)
- `logo.png` — Miami Dade College primary logo, sourced from mdc.edu

## Build notes

- Only external asset is Google Fonts (Open Sans), which matches MDC's live site.
- No JavaScript.
- Eduba branding is confined to the footer per the shared-chrome rule.
- Calendly link (`https://calendly.com/thecro-eduba/30min`) is live in three places: hero CTA, final CTA card, and author column.
